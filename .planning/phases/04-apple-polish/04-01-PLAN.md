# Phase 04-01 Plan: Emotional UX & Animation

**Goal**: Framer Motion을 활용하여 시각적 피드백과 감성적인 인터랙션을 강화하여 Apple 스타일의 완성도를 확보합니다.

## Must-Haves
- [ ] 시나리오 전환 및 UI 상태 변경 시 부드러운 레이아웃 전이 (`AnimatePresence`)
- [ ] 주요 수치(총 자산, 수익률 등) 업데이트 시 부드러운 숫자 카운팅 (`AnimatedCounter`)
- [ ] 자산 성장에 따른 마일스톤 달성 축하 애니메이션 (파티클 또는 배지)

## Task List

### 4.1.1 AnimatePresence Integration
- [ ] `src/App.tsx` 또는 메인 레이아웃에 `AnimatePresence` 적용
- [ ] 시나리오 변경 시 차트와 KPI Grid가 부드럽게 페이드인/아웃 또는 슬라이드되도록 구현
- [ ] `AdvancedSettingsSheet` 오픈/클로즈 시 감성적인 타이밍 조정 (Spring physics 적용)

### 4.1.2 AnimatedCounter Component
- [ ] `src/components/common/AnimatedCounter.tsx` (가칭) 컴포넌트 생성
- [ ] `framer-motion`의 `useSpring`과 `useTransform`을 활용하여 정밀하고 부드러운 숫자 애니메이션 구현
- [ ] `KPIGrid`의 모든 수치에 적용

### 4.1.3 Milestone Celebration UI
- [ ] 특정 자산 금액(예: 1억, 5억, 10억) 도달 시 트리거되는 애니메이션 로직 설계
- [ ] `canvas-confetti` 라이브러리 검토 또는 CSS 기반의 심플하고 고급스러운 축하 효과 구현
- [ ] 마일스톤 달성 시 KPI 카드에 시각적인 강조 효과(Glow 또는 Badge) 추가

## Success Criteria
1. 모든 수치 변화가 시각적으로 부드럽게 인지됨.
2. 화면 전환 시 UI 요소들이 서로 겹치거나 갑자기 사라지지 않고 유기적으로 연결됨.
3. 마일스톤 달성 시 사용자에게 성취감을 주는 시각적 피드백이 제공됨.
